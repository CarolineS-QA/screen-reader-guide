HMRC Accessibility Manual

NVDA

This guide will help you learn the basics of NVDA and some tips on how to test a service using it.

NVDA is only available on Windows.

As NVDA is an operating system level screen reading application, it means that it can be used for pretty much anything on the computer. That also means you can move away from the web page and into the address bar.

If you get lost just use your mouse to click back onto the browser. We are testing the websites, not your ability with the screen‐reader.

Installing NVDA

As NVDA is not built into the OS like Voiceover or Narrator, you will need to install it, so if you have not got it installed already, download NVDA for free from NV Access.

If you have already got NVDA installed it is worth checking there are no pending updates. You should be testing with the latest version.

Browser choice

Screen‐readers tend to work best with specific browsers. We currently recommend testing in Chrome, but check any new components or code which is specific to your service in Firefox as well.

How to start NVDA

You start NVDA just like any other program in Windows.

When you start NVDA it will appear as a tool tray icon near the date and time, you may need to expand the tool tray to see it.

Clicking on the icon will allow you to access the various settings. Whilst we recommend you leave most of the defaults to make consistent testing possible, there are a couple of changes which will make testing easier.

Change the Voice Synthesizer

The default voice NVDA uses is very robotic not especially pleasant to listen to for long periods. We suggest you change it to use a different synthesizer which sounds more human.

Go to “Preferences”, then “Settings”, then “Speech”. Try out some of the synthesizers from the drop-down. Different voices using that synthesizer are available from the voices drop-down below it.

I find the Windows OneCore Hazel voice to be quite good, but try a few out and find one which you are happy listening to.

Turn off Say All

When NVDA loads a new page, the defult setting is for it to read all of it out. As this can get annoying when testing, feel free to turn this off by going to “Preferences”, then “Settings”, then “Browse Mode” and unchecking the “Automatic say all on page load” option.

Turn off mouse tracking

If you hover over on anything now with the mouse, NVDA will announce what it is, and this will be the same for anything in the operating system as NVDA is an OS‐level screen‐reader. You shouldn't be using your mouse when testing however, and turning this feature off makes it easier to keep your place. To turn this off go to “Preferences”, then “Settings”, then “Mouse” and uncheck “Enable mouse tracking”.
