Basic navigation

This section is Voiceover for Mac

When you arrived at this page, you will have seen a black border laid around the whole page. This lets you know where the Voiceover focus is and it will follow you as you move around the page.

When a page loads

Voiceover reads out the title of the page first which is visible in the browser tab. The page title is very important as it helps users know which tab they are on and if that is the page they want to start exploring.

Sometimes when a page loads you might find the screen-reader starts reading part-way down the page. This can be down to the screen-reader remembering the last position on a previously visited page (trying to be helpful). However it is worth checking that it is not caused by any local code.

Check the first part of the page title and the page heading match when testing a service.

Right now the focus is on the whole page, so we want to move it inside the page so we can start testing.

From now on you should only use Voiceover to navigate the guide.

If you need to scroll down the page to read some instructions then feel free to do that until you have learned all the commands.

Moving into a page

To start interacting with the page use Control + Alt + shift + down arrow. You will also use this same combination to interact with other groups.

You should now be on the skip link at the top of the page, which should have become visible as it has focus.

Focus

As you move through a page and encounter focussable elements, you will see that the focus state appears on the element as normal.

You will notice however that when you move to other elements, paragraphs, labels, heading and the like, the focus state will remain on the previous link or input until you encounter another one at which point the focus state will move. This is normal.

When testing with screen-readers you need to pay attention to the on-screen indicator. In the case of Voiceover that is the black box around the element you are currently on.

Just be aware that whilst that focus state is active you can still accidentally trigger that element by not using the correct key combinations.

Moving through a page

Rather than clicking the skip link we're going to move through the page.

To do this you use the standard Voiceover modifier keys of Control + Alt and either right arrow to move forward or left arrow to move backwards

Use those three keys now to move through the page. You will notice a few things get read out which you might not have come across before. We'll talk about all of these in more detail later.

  • the black header gets announced as a “banner” and the page content gets announced as a “main”. These are called landmarks.
  • headings announce what level they are
  • this list was announced as a list and it said how many items were in it

Now move to the button below and use Control + Alt + space to activate it.

Commands used so far

Start/stop Voiceover
Command + F5
Start interacting with a page or group
Control + Alt + shift + down arrow
Move from one item to the next
Control + Alt + left/right arrows
Click a button or link
Control + Alt + space
